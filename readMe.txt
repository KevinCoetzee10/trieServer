Kevin Coetzee Compiax Trie Challenge

When using TAR directory:
1. Extract files to find compiax_trie folder
2. Open terminal within compiax_trie folder
3. Execute "node server.js" command
4. Open "http://localhost:3000" in browser

When using Git repository: 
1. Pull Git repository to a local folder
2. Open terminal within local folder
3. Execute "npm install" command
4. Execute "node server.js" command
5. Open "http://localhost:3000" in browser
